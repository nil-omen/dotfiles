# My dotfiles

This repo is for my dotfiles and scripts used after a new system install.


- To install my most used tools on [CachyOS](scripts/install_tools(CachyOS).sh), first make the scripts excutable:
```shell
chmod +x install_tools(CachyOS).sh
```
- Then run it:
```shell
./install_tools(CachyOS).sh
```

The same goes for [nerd fonts](scripts/nerd-fonts-install.sh) script
